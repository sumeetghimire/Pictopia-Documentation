Pictopia Documentation

A Simple Gallery Library for Images

Introduction

Pictopia is a simple gallery library that supports displaying images in a responsive grid layout. You can also view these images in a fullscreen modal with navigation, zoom, and thumbnail previews.

Required JavaScript

Make sure to include jQuery and the Pictopia script in your HTML:

<script src="https://cdn.jsdelivr.net/gh/sumeetghimire/pictopia/pictopia.js"></script>

How to Use

Follow these steps to integrate Pictopia into your website:

  1. Include the Pictopia library in your project (see the script below).
  2. Create a container with the class .Pictopia and add your images inside it.
  3. Initialize the library using Pictopia({thumbnail: true, displayPerRow: 3}).

Example Code

Here is a simple example showing how to display two images:

<div class="Pictopia">  //Pictopia class is required
    <img src="image1.jpg" alt="Image 1" class="max-w-full">
    <img src="image2.jpg" alt="Image 2" class="max-w-full">
</div>

<script>
  Pictopia({ thumbnail: true, displayPerRow: 3 });
</script>
